How To Easily communicate to Git Hub

Fist on upper Left search Your Repositories

Click New upper right

Click Create Repository

copy the URL of the web page

on your CLI type: git clone and paste the URL link a it will look like git clone https://yourlink.git hit enter

copy all your files to the folder that you clone from GitHub

and then on your CLI type git add .

on your CLI again type git commit -am 'your comment'

on your CLI again type git push
and then go to your git hub page and see your files
